Core Question

Does a higher immersion score align more clearly with replayability, public reception, or commercial potential? This prototype helps compare those signals without treating them as the same thing.

Score Matrix

Each game is scored across nine 1–5 metrics. The calculated immersion percentage converts the total out of 45 into a 0–100 score.

Website Prototype Concept

1. Import

The site starts with workbook data: players, games, score matrix, timeline responses, quiz questions, and quiz records.

2. Explore

Charts and tables let the researcher filter by game, player, and player type.

3. Collect

The response form stores additional notes in localStorage, so the prototype works without a backend.

4. Export

Exports create Excel-readable files that can be opened in Excel or imported into Google Sheets.

Export Data

Export all imported responses plus any locally added responses.
